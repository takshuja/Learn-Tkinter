Widgets

What are widgets?

Tkinter is a standard library in Python used to create Graphical User Interface (GUI) for desktop applications. With the help of tkinter, you can easily create any sort of GUI for desktop applications.
The best thing about Tkinter is that it is cross platform meaning the same code works on multiple platforms like Linux, macOS and Windows without installing anything else.
It is not a thin wrapper, but adds a fair amount of it's own logic to make the experience more pythonic.


Installation

If you are on Windows, while installing python check the td/tk and IDLE checkbox. However, if you have missed installing tkinter, you can easily do it using "pip" command like in Linux.

On Linux, open up the terminal, enter "pip install python3-tk" into the terminal and press enter. This will install the Tkinter module into your system.


Why use Tkinter?

You might be wondering there are so many Python libraries to create GUI. Why should I use Tkinter?
Well, don't worry. I have the answer. Here is a list of advantages of using Tkinter over any other GUI library.
  • Tkinter has a simple API which is really easy to learn as compared to other GUI toolkits.
  • Once you grasp the basics of, you can create a wide variety of applications using tkinter GUI toolkit.
  • It is cross-platform so the same code works on Linux, macOS and Window.
  • Tkinter widgets are rendered using native operating system elements so the applications built using tkinter look like they belong to the platform where they run.
  • Tkinter is more stable and flexible.
  • Tkinter has a large number of widgets which make it suitable for both simple and complex GUI development.
  • It is free to use.
  • It has a large community which is always ready to help.
