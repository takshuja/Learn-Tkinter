Layout Management


Layout managers or geometry managers are software components used in widget toolkits which have the ability to arrange graphical components in a particular manner without using distance units.

Layout Managers In Tkinter

In tkinter, we have three built-in layout managers namely pack, grid and place.
  • The pack layout manager organizes the widgets in horizontal and vertical boxes.

  • The grid layout manager places the widgets in a two-dimensional grid.

  • The place layout manager positions widgets using absolute positioning.
