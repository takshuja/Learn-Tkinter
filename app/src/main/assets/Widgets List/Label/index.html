Introduction to Label Widget

In Tkinter, the label widget is used to place text, images or bitmaps in a container-box. It is generally used to display information about other widgets in the GUI. The text inside a label can be changed at any time. You can span the text inside the label across multiple lines if needed. However, you can only use a single font at once.


  • Customising the label.

    • To customize a label in tkinter, we have a list of attributes some of which are listed below.


    Attribute Description
    Anchor Align the text inside the widget.
    bg Background color of entry.
    bd Border-width of the entry. By default, it is 2 pixels.
    bitmap set the bitmap or image and display it in the label
    cursor Set the cursor type of the mouse pointer.
    fg Text color of the label.
    font Font of the text to be used for label.
    height Set the height of the label.
    image Image to be dispalayed as the label.
    justify Justify/align the text. It's values are left-justified, right-justified, center.
    padx The padding area between the button and it's sides on the x-axis. The default value is 1.
    pady The padding area between the button and its sides on the y-axis. The default value is 1.
    relief Set border type. Some of the values are GROOVE, RAISED, RIDGE, SUNKEN, FLAT.
    text Set the text of the label.
    textvariable Sets the variable that will be displayed as text. When the variable changes, the label text also changes.
    underline Underline the character of text on that index. If it is set to -1, no character will be underlined.
    width Width of label in characters.
    wraplength Break the label text into multiple lines where each line has number of characters specified to this option.
