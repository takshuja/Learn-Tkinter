Introduction to scale Widget

The scale widget is used to implement the graphical scale to the python application so that the user can slide through a range of values and select the one among them.

The maximum and minimum values of the scale can be controlled along with the resolution of the scale .



Customising the scale .

To customize a scale in tkinter, we have a list of attributes some of which are listed below.


Attribute Description
activebackground Background color of the scale when mouse hovers it.
bg Background color of the scale .
fg scale 's foreground/text color.
bd Border-width of the scale .
command Set the callback function of the scale . It gets called everytime you move the scale .
cursor The mouse pointer is changed to the assigned cursor type.
digits To be implemented later.
font Font of the text to be used for scale 's label.
from_ It is used to represent one end of the widget range. It is the starting value of the range.
height Set the height of the scale .
highlightcolor Focus highlight color when scale has focus.
image Image to be displayed on scale instead of text.
justify Justify/align the text. It's values are left-justified, right-justified, center.
orient It can be set to HORIZONTAL or VERTICAL depending upon the requirement.
padx The padding area between the scale and it's sides on the x-axis.
pady The padding area between the scale and its sides on the y-axis.
relief Set border type. Some values are GROOVE, RAISED, RIDGE, SUNKEN, FLAT
state Set this option to DISABLED to make it responsive or to NORMAL to make it responsive.
underline Underline the character on that index. If it is set to -1, no character will be underlined.
width Width of scale in characters( if displaying text) or in pixels (if displaying an image).
wraplength Specify maximum line length in pixels for word-wrapping.
