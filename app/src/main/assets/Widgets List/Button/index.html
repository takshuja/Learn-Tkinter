Introduction to Button Widget

Buttons are standard widgets in a GUI represent a clickable item in the application. They come with the default Tkinter library. We can configure the look of the button according to our requirements.

To make a button do something, you associate a function or method with a button. This is called as command binding in tkinter. This function or method is called as the callback function. The callback function is called when a button is clicked.

Note: A button can only display text in a single font. However, the text of a button can span multiple lines.


Customising the button.

To customize a button in tkinter, we have a list of attributes some of which are listed below.


Attribute Description
activebackground Button's background color when it gets clicked.
activeforeground Button's foreground color when it gets clicked.
bg Button's background color.
fg Button's foreground/text color.
bd Border-width of the button.
command Set the callback function.
font Font of the text to be used for button's label.
height Set the height of the button.
highlightcolor Focus highlight color when button has focus.
image Image to be dispalayed on button instead of text.
justify Justify/align the text. It's values are left-justified, right-justified, center.
padx The padding area between the button and it's sides on the x-axis.
pady The padding area between the button and its sides on the y-axix.
relief Set border type. Some of the values are GROOVE, RAISED, RIDGE, SUNKENFLAT
state Set this option to DISABLED to make it responsive or to NORMAL to make it responsive.
underline Underline the character on that index. If it is set to -1, no character will be underlined.
width Width of button in characters( if displaying text) or in pixeld (if displaying an image).
wraplength Specify maximum line length in pixels for word-wrapping.
