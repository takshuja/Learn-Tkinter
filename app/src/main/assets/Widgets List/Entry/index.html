Introduction to Entry Widget

In Tkinter, the entry widget is used to accept a single-line text value from the user.


  • Customising the entry widget.

    • To customize an entry in tkinter, we have a list of attributes some of which are listed below.


    Attribute Description
    bg Background color of entry.
    bd Border-width of the entry. By default, it is 2 pixels.
    cursor Set the cursor type of the mouse pointer.
    exportselection If you select some text inside the entry, it will be automatically copied to the clipboard. To disable this action, set the value of exportselection to 0
    fg Text color of the label.
    font Font of the text to be used for label.
    highlightbackground Set the color focus highlight when the button has focus.
    justify Justify/align the text. It's values are left-justified, right-justified, center.
    relief Set border type. Some of the values are GROOVE, RAISED, RIDGE, SUNKEN, FLAT.
    selectbackground Background color of the text when it is selected.
    selectborderwidth The width of the border of selected text.
    selectforeground The foreground color of the text when it is selected.
    show It is used to show the entry text of some other type instead of the entered string.
    To show password as '*', we set the value of show to '*'.
    textvariable Sets the variable that will be used to retrieve the text from the entry.
    xscrollcommand Link the entry widget with the scrollbar in case we want the user to enter more text than actual width of the widget.


  • Tkinter Entry Methods.

    • Tkinter entry has various methods to configure the text written inside the widget. Some of the methods are listed below.


    Method Description
    delete(first, last=none) It is used to delete characters in the specified index inside the widget.
    get() It is used to get the text inside the widget.
    insert(index, string) It is used to enter the string before the character at the given index.
