Introduction to checkbutton Widget

A checkbutton is a graphical widget that permits a user to select one or more options from a limited number of choices. A caption describing the meaning of the checkbutton is usually shown adjacent to the checkbutton. A checkbutton has two states i.e: on and off. The state of checkbutton can be changed by clicking the mouse on the box, by clicking the caption of the checkbutton or by pressing the space-bar.

Tkinter checkbutton attributes.
Here is the list of commonly used checkbutton options.


Attribute Description
activebackground Background color of checkbutton when it is under the mouse pointer.
activeforeground Foreground color when it is under the mouse pointer.
bg Background color of the checkbutton's label.
fg Text color of the checkbutton's label.
bd Border-width of the checkbutton. By default it is 2 pixels.
command Call the callback function when the user changes the state of checkbutton.
cursor Set the cursor type of the mouse pointer when it is over the checkbutton.
disableforeground Set the text color of a disabled checkbutton.
font Font of the text to be used for the label of checkbutton.
height Set the height of the checkbutton(in terms of number of lines). The default value is 1.
highlightcolor Focus highlight color when checkbutton is under focus.
image Image used to represent the checkbutton.
justify Justify/align the text. It's values are left-justified, right-justified, center.
offvalue Set the value of checkbutton when it is off (unchecked). By default it is 0. It's values can be 0 or 1
onvalue Set the value of checkbutton when it is on (checked). By default it is 1. It's values can be 0 or 1
padx The vertical padding of the checkbutton.
pady The horizontal padding of the checkbutton.
relief Set border type. Some of the values are GROOVE, RAISED, RIDGE, SUNKENFLAT
selectcolor The color of the checkbutton when it is set.
selectimage The image is shown on the checkbutton when it is set.
state Set this option to DISABLED to make it responsive or to NORMAL to make it responsive.
underline Underline the character on that index. If it is set to -1, no character will be underlined.
variable This option represents the associated variable that tracks the state of the checkbutton.
width Width of button in characters( if displaying text) or in pixels (if displaying an image).
wraplength Specify maximum line length in pixels for word-wrapping.


  • Tkinter checkbutton methods.

    • The commonly used methods for this widget are listed below.


    Method Description
    deselect Clear checkbutton selection.
    flash Flash the checkbutton between it's active and normal colors.
    invoke Get the same action as when user clicks the checkbutton.
    select Select the checkbutton.
    toggle Invert the state of checkbutton.
